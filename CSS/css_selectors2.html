Article on CSS Selectors

CSS Selectors are patterns used to select the HTML elements we want to style. Rather than writing inline CSS for each element and same style for other elements we can use CSS selectors to minimise the duplication of CSS styling code. Lets try to understand the different types of selectors below:

Universal selector

Universal selector allows to style all the elements on the HTML page. An asterick (*) is used for selecting all elements. With the code below, the backgound and text color of the entire HTML page changes to specified colors.

*{
background-color:#414848;
color:#7b929c;
}

Individual selector

Individual selectors allows to style the HTML elements we want. Each element can have different style. The code below all the headings with h3 and p tag will have the color changed

h3{
color:#9881ab;
}
p{
color:#CAD5E2;
}

Class and ID Selector

Class Selector allows to create a CSS class with the styles we want, these classes can be used for any HTML elements.

.highlight{
background-color: yellow;
color:black;
}
<p class="highlight">Class Selector allows .......</p>

ID Selector allows to create a set of CSS styling, these id's should be uniquely used in HTML i.e, no two HTML elements should use the same ID, as ID should be unique for all HTML elements.

#ident{
background: white;
color:red;
}
<p id="ident">ID Selector allows ......</p>

AND Selector

AND Selector allows to write a conditional styling, that is if we want to style a HTML element which has all the properties we specify then only the style is applied. Let us understand it better with an example:
In the below code we see that 'Hello' is a paragragh and span and also it has a class 'highlight'

<p><span class="highlight">Hello</span><p>

Now lets write an AND Selector CSS

p span.highlight{
background-color: red;
color:white;
}

The result should display Hello with red background as below:

Hello

Combined Selector

Combined Selectors allows same styling for any specified HTML elements, that is more than one HTML element type can have same styling.

h1, code{
color:#d9ce98
}

Inside an Element Selector

Inside an Element Selector allows to identify the nested elements and styles them.
In the below code we are styling the code element which is inside div element, div element is inside body element:

body div code{
display:block;
width:500px;
color:#414848;
background-color: white;
}

Direct Child Selector

Direct Child Selector allows to style the direct child elements, that is if the element is directly inside of the other element then it is used.

div>h3{
color:aqua;
}

Sibling Selector Selectors

Sibling Selector Selectors allows to style the just next element of the element using the sibling class.

.sibling + li {
display:inline;
color:black;
background-color: pink;
}
  • First elementin list using sibling class
  • Second element of the list next to the element using sibling class
  • Third element.

Before After Selectors

Before After Selectors are Pseudo Selectors, these are used to add cosmetic styling before and after the HTML elements
Few scenarios where we can apply these, before all the list if we want to add an image before the lists we can use ::before selector.
We can use after selector if we want to insert any emoji or link emoji after the hyperlink.
